```bash [Docker]
# Pull the latest image
docker pull sigp/anchor:latest

# Run the container
docker run --rm -it sigp/anchor:latest --help
```

```bash [From Source]
# Clone and build
git clone https://github.com/sigp/anchor
cd anchor
# Make the binary
make
# Make sure ~/.cargo/bin is in your $PATH
anchor --help
```

```bash [From Release]
# Specify the platform i.e x86_64-apple-darwin (for apple) x86_64-unknown-linux-gnu.tar.gz
wget https://github.com/sigp/anchor/releases/download/v0.2.0/anchor-<platform>.tar.gz
# Extract the file
tar -xvf anchor-<platform>.tar.gz
# Make it executable
chmod +x anchor
# Run it
./anchor --help
```
